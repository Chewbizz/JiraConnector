JiraConnector
========================================

JiraConnector is a fork of the 'Atlassian Connector for Eclipse' plugin

It provides Jira/Bamboo integration withing Eclipse.

Works with Bamboo 2.7+ and Jira 6.0+

Requires Java 11, Eclipse 2021-12 or later
